## 🔐 Cybersecurity Focus

Security isn't an afterthought in my projects — it's baked into the architecture:

| Area | What I've built / applied |
|------|--------------------------|
| **Cryptography** | RSA-PSS signatures, HMAC message authentication, salted password hashing |
| **Auth systems** | JWT-based auth with role-based access control (RBAC) across multiple user roles |
| **TLS/SSL auditing** | Python-based certificate security auditor with 10+ automated checks |
| **API security** | Swagger/OpenAPI docs generated via FastAPI — gated in production to prevent recon |
| **Biometric auth** | `local_auth` integration in Flutter for device-level identity verification |
| **Secret management** | ENV-variable-driven config across all production deployments; no secrets in version control |
| **Incident response** | Secret purging from git history using `git filter-repo` |
| **Secure coding** | CORS hardening, rate limiting, input validation at the API layer |
